# react-geolocation-element

A lightweight React wrapper for the `<geolocation>` element

## Installation

```bash
bun add react-geolocation-element
```

## Usage

```tsx
import 'react-geolocation-element/styles.css';
import { Button } from 'react-geolocation-element';

function App() {
  return <Button>Click me</Button>;
}
```
